PDK Fritzing Library
=====================

This package installs the complete PDK parts library and adds a dedicated
"PDK Access Control" bin to Fritzing.

INSTALL

1. Close Fritzing.
2. Double-click "Install PDK Fritzing Library.cmd".
3. Reopen Fritzing if it does not start automatically.

The installer works in your user Documents folder and does not require
administrator rights. Existing library files are backed up before changes
are made.

INCLUDED PARTS

- PDK Red Max Enclosure
- Altronix eFlow102NB
- PDK Red 1 Expander (R1E)
- PDK Red 2 Expander (R2E)
- PDK Red 4 Expander (R4E)
- PDK Red Aux 8 (A8)
- PDK WiMAC Module
- PDK Red Series PoE Module

The WiMAC and PoE modules are sized from PDK's installation graphics so they
match the controller sockets and mounted footprints shown in the manuals.

UNINSTALL

Double-click "Uninstall PDK Fritzing Library.cmd".
