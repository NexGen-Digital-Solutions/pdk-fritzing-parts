# PDK Red Series PoE Module

- Fritzing module ID: `PDK_Red_Series_PoE_Module_v1`

This version is sized from PDK's Red 2 installation graphic, where both module headers are shown aligned with the controller's PoE Out and PoE In sockets.

- Footprint: **1.7231 × 1.0194 inches**
- Contacts: **10**, six for PoE Out and four for PoE In
- Electrical pin names remain generic until a verified factory pinout is available

The earlier 2.54 mm assumption made the module too large for the controller artwork. This revision follows the official mounted footprint instead.
