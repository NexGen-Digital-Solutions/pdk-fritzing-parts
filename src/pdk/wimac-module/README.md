# PDK WiMAC Module

- Fritzing module ID: `PDK_WiMAC_Module_v1`

This version is sized from PDK's Red 4 installation graphic, where the module is shown installed over the controller's twin vertical sockets.

- Footprint: **0.6923 × 1.1122 inches**
- Orientation: antenna connector up, ready to place over the R4E sockets
- Contacts: **20**, ten per side
- Electrical pin names remain generic until a verified factory pinout is available

The earlier 2.54 mm assumption made the module much too large. The official placement graphic shows a substantially finer connector system.
