# Release 1.0.0

## Initial release

The initial release includes eight parts for documenting and planning PDK Red
Series access-control panels. The WiMAC and Red Series PoE modules are sized
from PDK's installation graphics, which show smaller connector systems than a
standard 2.54 mm grid.

The corrected dimensions are:

| Module | Footprint | Contacts |
|---|---:|---:|
| PDK WiMAC Module | 0.6923 × 1.1122 in | 20 |
| PDK Red Series PoE Module | 1.7231 × 1.0194 in | 10 |

The WiMAC part is shown in its installed orientation, with the antenna
connector facing upward.
